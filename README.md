# yanglabkit

Yang Lab skill set for AI agents. A focused collection of skills tailored to the
Yang Lab's own research workflows — **not** a general-purpose research toolkit.

The broad research lifecycle (literature search, citation management, LaTeX
checks, manuscript critique, and so on) is already well served by other skill
sets. `yanglabkit` deliberately stays narrow: it covers only the specific,
recurring needs of how *we* work that aren't handled well elsewhere.

Skills use the portable `SKILL.md` convention, so they're usable by any AI agent
that supports it — [Claude Code](https://claude.ai/code) and others alike, not
tied to a single harness.

> **Status**: scaffold only. No skills implemented yet.
